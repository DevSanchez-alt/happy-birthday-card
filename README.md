# Feliz Cumpleaños, Suleyma 

Una tarjeta de cumpleaños interactiva hecha con HTML, CSS y un toque de JavaScript — diseñada con cariño, estilo romántico pastel.

## Ver la tarjeta

Abre `index.html` en tu navegador, o visita la versión publicada en GitHub Pages (una vez activado).

## Características

- Sobre animado con sello que se abre al hacer clic
- Confetti al abrir la tarjeta
- Pétalos cayendo de fondo
- Velita con llama animada
- Música de fondo (se activa al abrir la tarjeta)
- Botón para pausar / reanudar la música
- Diseño responsive (se ve bien en móvil y escritorio)
- Tipografía elegante (Cormorant Garamond + Dancing Script)

## Cómo usar

1. Clona o descarga este repositorio
2. Abre `index.html` en cualquier navegador moderno
3. Haz clic en el sello rosa para abrir la sorpresa ♡
